This Java Enterprise Edition 6 Web application has the following features.

  • Clean semantic HTML5 mark-up
  • CSS3 Responsive Web Design with mobile first
  • Modernizr, jQuery, polyfill, and plug-in JavaScript libraries
  • Utilizing GroundworkCSS, bootstrap-sass, CSS3, SASS, and JavaScript framework that makes responsive design easy
  • Server side includes (SSI) with the Apache Tomcat Java SSI Servlet
  • Dynamic content with Java API for RESTful Web Services (JAX-RS) 1.1
  • Java Persistence API (JPA) 2.1 Entity Java objects
  • The Jackson Java JSON-processor is utilized to extend JAX-RS, along with its annotations
  • Server side service delegation prior to SSI processing based on the Delegation Design Pattern
  • Internationalization (i18n) and localization feature
  • Production mode features for browser performance
