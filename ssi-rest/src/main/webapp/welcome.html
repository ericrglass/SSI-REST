This Java Enterprise Edition 6 Web application has the following features.

  • Clean semantic HTML5 mark-up
  • CSS3 Responsive Web Design with mobile first
  • Modernizr, jQuery, Knockout , polyfill, and plug-in JavaScript libraries
  • Utilizing GroundworkCSS, bootstrap-sass , CSS3, SASS, and JavaScript frameworks that makes responsive design easy
  • Templates and reuse of HTML5 mark-up with Server side includes (SSI) using the Apache Tomcat Java SSI Servlet
  • Dynamic content is provided with JSON generated on the server side using Java API for RESTful Web Services (JAX-RS) 1.1
  • The database connections and data processing is managed on the server side using Java Persistence API (JPA) 2.1 and plain old Java objects as the Entities
  • The Jackson Java JSON-processor is utilized to extend JAX-RS, along with its annotations, in the generation of JSON from plain old Java objects
  • Server side service delegation prior to SSI processing is based on the Delegation Design Pattern
  • Internationalization (i18n) and localization features for content and CSS3 styling
  • Production mode features that help improve server side response and Web browser rendering performance
